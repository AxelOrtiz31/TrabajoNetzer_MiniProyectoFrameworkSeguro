# 📋 Módulo de Gestión de Tareas - Django

Sistema de gestión de tareas con autenticación, roles de usuario y sesiones seguras desarrollado en Django como actividad académica.

## ✨ Características

- 🔐 **Autenticación de usuarios**: Sistema de login/logout protegido
- 👥 **Roles diferenciados**: Administradores (staff) vs Usuarios normales
- ✅ **Validación de formularios**: Validaciones tanto en modelo como en formulario
- 🛡️ **Protección CSRF**: Implementada en todos los formularios POST
- 🔒 **Sesiones seguras**: Configuración con cookies HttpOnly, SameSite y expiración
- 📱 **Interfaz responsive**: Diseño con Bootstrap 5

## 🛠️ Tecnologías utilizadas

- **Backend**: Django 4.x
- **Base de datos**: SQLite (desarrollo)
- **Frontend**: HTML5, Bootstrap 5
- **Autenticación**: Sistema de auth de Django

## 📋 Prerrequisitos

- Python 3.8 o superior
- pip (gestor de paquetes de Python)
- Git (opcional)
